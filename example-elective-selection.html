💡 Sistem Nasıl Çalışır?

  • Müsait Seçmeli Dersler: Henüz hiç sınıfa atanmamış, seçilebilir durumda olan dersler
  • Atanmış Seçmeli Dersler: Bir öğretmen tarafından bir veya birden fazla sınıfa atanmış dersler
  • Çakışma Önleme: Bir ders atandığında, aynı seviyedeki diğer öğretmenlerin listesinden otomatik olarak kaldırılır
  • Kaynak Yönetimi: Sınırlı öğretmen kaynakları optimum şekilde kullanılır
  • Gerçek Zamanlı Güncelleme: Atamalar anında sisteme yansır

🔧 Backend API Fonksiyonları

Aşağıdaki API çağrıları frontend'de kullanılabilir:

// Müsait seçmeli dersleri getir
window.electronAPI.lesson.getAvailableElectives(grade)

// Atanmış seçmeli dersleri getir
window.electronAPI.lesson.getAssignedElectives(grade)

// Bir seçmeli dersin müsait olup olmadığını kontrol et
window.electronAPI.lesson.isElectiveAvailable(lessonId, grade)

// Belirli bir sınıf için müsait seçmeli dersleri getir
window.electronAPI.lesson.getAvailableElectivesForClass(classId)

// Seçmeli dersleri durumlarıyla birlikte getir
window.electronAPI.lesson.getElectivesWithStatus(grade)
